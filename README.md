# Setup
```bash
# Установка storybook для тестирования отдельных UI-компонентов
npm create storybook@latest

# Повторный запуск storybook для тестирования отдельных UI-компонентов
npm run storybook

# Запуск проекта (из корневой директории)
npm run dev
```
